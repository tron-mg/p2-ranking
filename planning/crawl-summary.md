## What the site is doing (observed IA pattern)
- **Commercial-investigational hub model**: ranking pages are the main SEO + conversion drivers.
- **Dual intent tracks**:
  - Beauty-focused ranking intent (uroda): do picia / rybi / w proszku
  - Joint-focused ranking intent (stawy): dedicated ranking and scoring logic
- **Methodology trust layer**:
  - “Jak oceniamy” and scoring explanations used as E-E-A-T support.
- **Educational cluster layer**:
  - Evergreen explainers around mechanisms, forms, dosage timing, effects, and terminology.
- **Strong cross-linking behavior**:
  - Rankings repeatedly link to each other and to explanatory articles.

## Key content templates identified
1. **Ranking page template**
   - Intro + purpose + “switch ranking” CTA
   - Scoring framework (point categories)
   - Buyer education sections (what it is / how to choose / effects / risks)
   - Repeated anchors (“back to ranking top”)
2. **Educational explainer template**
   - Definition-first
   - Mechanism/how-it-works
   - Practical guidance
   - Internal links to relevant rankings
3. **Methodology template**
   - How evaluation works
   - Category-specific scoring methods
   - FAQ on fairness/transparency

## Observed strengths
- Clear topical authority around collagen sub-intents.
- Good internal bridge between category rankings and educational pages.
- Frequent scientific framing improves trust posture.

## Gaps/opportunities for fast-v1 IA
- Add explicit **pillar → cluster breadcrumbs** for all ranking pages.
- Standardize **intent labels** (e.g., uroda vs stawy) across headings/meta.
- Build stronger **decision pathways** (“which ranking to open first?”) from homepage and cluster pages.
- Normalize **cross-links to sibling rankings** near top of each ranking page.
